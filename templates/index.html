Satellite Image Brightness Normalizer

This tool normalizes the brightness of satellite images to a global average intensity. Upload a ZIP file containing grayscale PNG images to begin.

Input Requirements:
  • A ZIP file containing 10 grayscale PNG images
  • Each image should be 256x256 pixels, 8-bit depth (0-255)
  • Files should be named image1.png through image10.png
Output:
  • 10 normalized grayscale PNGs with same dimensions
  • Each image will have an average intensity within ±1 of the global average
  • Processing completes in under 10 seconds
